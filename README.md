# Eventify 🎉

Eventify is a MERN stack Event Management System.

## 🚀 Live Demo

👉 [View Website](https://smruthykrishna.github.io/Eventify)

## 🛠️ Tech Stack

- MongoDB
- Express.js
- React.js
- Node.js

## 📦 How to Run Locally

1. `cd frontend`
2. `npm install`
3. `npm start`

---

Backend runs with:

1. `cd backend`
2. `npm install`
3. `npm run dev`
